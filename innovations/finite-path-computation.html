Finite Path Integral Computation

Quantum gravity without infinities

Executive Summary

Standard quantum field theory requires renormalization to handle divergent integrals. This works for QED but fails for gravity. RDT's finite configuration space Ω makes path integrals finite sums — no infinities, no renormalization needed. This could enable direct computation of quantum gravity effects.

The RDT Insight

Feynman's path integral sums over all possible paths. In continuous spacetime, this sum is infinite and often divergent. In finite Ω, the sum is finite: $\langle \sigma_f | \sigma_i \rangle = \sum_{\gamma \in \Gamma} e^{iS[\gamma]/\hbar}$ — a finite sum over real paths, no divergences, no renormalization.

Theoretical Foundation

2.1 The Renormalization Problem

Quantum field theory calculations often produce infinite results:

$$\int_0^\infty \frac{d^4k}{k^2} = \infty$$

Renormalization techniques subtract these infinities systematically. For QED, this works spectacularly (predictions match experiment to 12 decimal places). For gravity, renormalization fails — gravity is "non-renormalizable," meaning infinities cannot be consistently removed.

2.2 Finite Ω Resolves the Problem

If Ω is finite with $|Ω| = N \approx 10^{10^{122}}$ configurations, the path integral becomes:

$$\langle \sigma_f | \sigma_i \rangle = \sum_{\gamma: \sigma_i \to \sigma_f} e^{iS[\gamma]/\hbar}$$

This is a finite sum. Enormous, yes (paths can have up to N steps), but finite. No UV divergences because there's a natural cutoff at the Planck scale (the minimum Ω-distance). No IR divergences because Ω is bounded.

2.3 Computational Tractability

Despite being finite, direct summation is impossible — $N \approx 10^{10^{122}}$ is unimaginably large. But standard techniques apply:

  • Monte Carlo sampling: Sample paths weighted by $|e^{iS/\hbar}|$
  • Importance sampling: Focus on paths near classical solutions
  • Renormalization group: Coarse-grain and flow to observable scales
  • Tensor networks: Exploit structure in path summation

The key advantage: these techniques now have rigorous justification. There's an actual finite sum we're approximating, not a formal divergent integral.
